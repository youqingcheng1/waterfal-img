瀑布流布局
====

>基于vue 的瀑布流布局

### window.getComputedStyle()方法
获取样式，和element.style 不同，window.getComputedStyle()只支持读取属性值不支持写入
通过window调用
```
window.getComputedStyle(element[,pseudo-element]);
```

### replace()
replace() 方法用于在字符串中用一些字符替换另一些字符，或替换一个与正则表达式匹配的子串。
